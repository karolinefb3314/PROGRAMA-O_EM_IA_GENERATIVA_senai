# README — IA Generativa

## 📌 Sobre o Projeto

Este repositório reúne conteúdos, aulas e materiais relacionados à **Inteligência Artificial Generativa**, abordando desde fundamentos de IA e Machine Learning até aplicações modernas com modelos generativos, visão computacional e processamento de linguagem natural.

O objetivo é fornecer uma base prática e teórica para estudantes, desenvolvedores e entusiastas que desejam aprender a construir soluções utilizando IA moderna.

# 🧠 O que é IA Generativa?

IA Generativa é um ramo da Inteligência Artificial capaz de criar novos conteúdos, como:

* Textos
* Imagens
* Áudios
* Vídeos
* Código

Utilizando modelos avançados de Deep Learning, como:

* Transformers
* GANs
* Diffusion Models
* Large Language Models (LLMs)
